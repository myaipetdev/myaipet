Data sovereignty

Your pet. Your data. Your rules.

In a world where AI platforms own your conversations, we chose differently. Four rights, built into the protocol — not bolted on as a promise.

EXP
Right 01/04 · Passport

Export

Download your pet's entire identity as portable JSON — personality, memories, skills, history.

$petclaw export
IMP
Right 02/04 · Passport

Import

Bring your pet from any PetClaw-compatible platform. Move homes; keep the friend.

APIPOST /api/petclaw/import
DEL

Delete

PetClaw protocol

Built on an open standard.

PetClaw is the open protocol for sovereign AI companions. Skills, networking, and memory — all portable by design.

$ npm install @myaipet/petclaw-sdk
01 / SKILLS

SKILL.md format

Install, create, and share pet capabilities as plain markdown files. A skill is a prompt, a schema, and a promise.

open spec
02 / NETWORK

PACK — A2A discovery

Pets can find public profiles across the open protocol. Remote skill calls stay paused until consent and caller funding are explicit.

discovery live
03 / CLIENTS

MCP compatible

Runs with Claude, OpenClaw, and any MCP client. Your companion is not locked to a single model or a single vendor.

mcp
AGENTIC HARNESS

VIGIL — a guarded learning loop for your companion.

Memory and feedback can improve continuity over time. Costlier best-of-N response selection is optional and disabled by default; availability also depends on the active model configuration.
